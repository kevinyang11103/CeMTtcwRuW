# 项目概述 python1173

本项目是一个基于Web的信息加密解密系统，采用Python语言及Django框架进行开发，数据库使用MySQL。系统遵循前后端分离的架构模式，以B/S和MVC为基础进行设计。

## 技术栈

- **架构**: B/S、MVC
- **开发环境**: 
  - PyCharm
  - Python 3.x
  - MySQL 5.x
- **技术框架**: 
  - Python
  - Django
  - MySQL

## 功能模块

### 前台系统

- **用户登录**: 用户登录系统，进行身份验证。
- **系统信息展示**: 展示系统的基本信息，如当前用户数、算法数等。
  - 当前用户数
  - 加密解密算法种类
- **在线加密解密**: 用户可以输入信息，选择加密或解密算法，进行信息加密或解密操作。
- **用户信息维护**: 用户可以查看和修改个人信息。

### 后台管理系统

#### 管理员端

- **算法管理**: 管理员可以添加、修改和删除加密解密算法。
- **用户管理**: 管理员可以查看、添加、修改和删除用户信息。
- **系统信息管理**: 管理员可以修改系统信息，如公告内容等。

## 系统角色

- 用户：可进行登录、信息加密解密以及个人信息维护。
- 管理员：负责算法管理、用户管理及系统信息管理。
